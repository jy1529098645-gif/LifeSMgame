# 荒诞人生模拟器 · Web 原型 (v0.3)

跨世代、**行动驱动、按周推进**的 Roguelite 人生模拟器：
- 先选**性别 + 出生年代/年份**，再用**骑砍式成长抉择**捏出一个人（所有开局总属性相等，平衡）。
- 每个大阶段进入时有**一个明确选择**（青年岔路口/事业十字路口…）；**留学等抉择会占去人生好几年**（逐年叙事）。
- **风口永不明示**，靠新闻自己嗅；时间线延伸到 2055（AI→机器人→聚变→银发→脑机→太空），不同世代撞上不同未来。
- **恋爱有前因后果**：先遇见心动的人 → 追求养好感 → 表白，不会凭空在一起。
- 从 18 岁起**按周**生活：每周有固定的**可支配小时数**，各行动按真实耗时（上班 40h、创业 48h、刷手机 4h…）消耗，做多少全看时间上限。
- 行动触发**视觉小说式事件，多级分支**：点一个大方向选项 → 展开下一层更细的选择，像对话树一样把故事捏出来。
- **多维属性 + 阶级决定叙事走向**（同种事件穷富两种世界、NPC 态度截然不同）；**健康只决定死亡概率**。
- **风口隐藏**，靠每周新闻自己推断、押注赛道；**结局每周概率判定**（上市/破产/移民/猝死/善终…）。

创业是主线目标，但卖点是亲手塑造一段荒诞人生。Web 原型验证玩法与量产内容；**正式版在 Unity 开发**（美术/特效/打包）。

## 运行

直接双击 `prototype/index.html` 即可（纯前端，`file://` 可运行，无需服务器）。
或在 `prototype/` 下起任意静态服务器，例如：`python -m http.server` 后访问 `http://localhost:8000`。

## 怎么玩

1. 选出身（决定起点、解锁的人生轨道、专属梗）。
2. 每个阶段（≈3 年）把有限的「时间·精力」点数分配到并行轨道：打工 / 副业 / 创业 / 学习 / 留学 / 人脉 / 恋爱 / 养生。
3. 推进后触发一个叙事抉择事件。
4. 健康越差、暗伤越多，死亡概率越高。死亡时结算六轴成绩 + 死法 + 人生回顾，「再开一把」。

## 项目结构

```
prototype/
  index.html   入口
  style.css    主题样式
  content.js   ★内容数据（背景/轨道/事件/暗伤/死法/称号）—— 加内容只改这里
  engine.js    规则引擎（回合循环/死亡判定/结算/渲染）
docs/
  GAME_DESIGN.md   ★完整设计文档（任何 AI/协作者先读这份）
```

## 给协作者 / AI 的话

**动手前先读 [`docs/GAME_DESIGN.md`](docs/GAME_DESIGN.md)**——它是单一事实来源，包含维度定义、咬合规则、死亡公式、事件编写规范、数据 schema 和 Unity 移植说明。扩展内容（背景/轨道/事件/结局）只需编辑 `content.js`，无需改引擎。
