# 创业人生模拟器 · 成都 2026（Web 原型 v0.5）

**「00 后职场沉浮」**——一座可点击的成都城市俯瞰图 + 按周推进的人生模拟器。
你从大三的毕业焦虑开始，海投被拒、试用期规训、通勤房租、职场荒诞，一路熬出自己的认知与人脉，
最终在「离职创业」的十字路口下海，让公司活过下一个冬天。**创业是终局，但卖点是亲手活过这一段沉浮。**

> 🎮 在线试玩：<https://web-production-a7eee.up.railway.app/>

---

## 玩法一览

- **城市俯瞰图主界面**：不再是一排抽象按钮，而是一张成都城市图。点区域 → 放大 → 点设施做事。
  求职去公共就业服务中心、上班去高新区写字楼 / 天府软件园、压力大去公园、没钱去便利店、累了回出租屋。共 15 个区域。
- **按周推进、行动驱动**：从大三起按周生活，每周有固定可支配小时数；上班自动吃掉大把时间，
  剩下的时间分配给副业 / 学习 / 求职 / 人脉 / 恋爱 / 养生。做多少全看时间上限。
- **7 阶段主线脊柱**：大三探索 → 求职准备 → 初入职场 → 职场沉浮 → 创业契机 → 离职决断 → 创业生存。
  每个阶段有明确的任务目标和关键节拍（beats），把零散事件串成一条能走通的主线。
- **VN 式叙事事件、多级分支**：行动触发视觉小说式事件，点一个大方向 → 展开下一层更细的选择，像对话树一样把故事捏出来。
- **多维属性 + 阶级决定走向**：6 维属性 + 人脉 / 声誉 + 出身阶级共同决定哪些事件出现、NPC 态度、选项成败。
- **手机 / 电脑桌面（可交互）**：
  - 📱 **绿泡泡**（仿微信：对话 / 群聊 / 红包 / 转账 / 自定义备注）、**老大直聘**（线上投递 + 面试）、
    **淘粑**（淘宝式商品瀑布）、**抖声**（竖屏短视频信息流）、**穷途熊熊**（理财）、相册、短信。
  - 💻 电脑端是真·桌面客户端：多标签页浏览器 + 绿泡泡桌面版等。
- **生存压力真实落地**：月度账单明细、房租通勤、资产缩水，钱耗光会饿死；健康只决定死亡概率，不是用来换钱的资源。
- **结局是概率的**：每周判定一次（上市 / 破产 / 猝死 / 善终…），由条件 + 概率触发，非必然。

## 运行

纯前端，经典 `<script>`，`file://` 可直接跑：

- 直接双击 `prototype/index.html`。
- 或在 `prototype/` 下起静态服务器：`python -m http.server` 后访问 `http://localhost:8000`。
- 项目根目录的 `python main.py`（零依赖标准库服务器，带 no-cache 头）也可一键起本地 / 部署用服务。

## 项目结构

```
StartupLifeSim/
├── main.py            部署用静态服务器（serve prototype/，读 $PORT，no-cache）
├── Procfile / railway.json / requirements.txt / runtime.txt   Railway 部署配置
├── DEPLOY.md          部署说明
└── prototype/         游戏本体
    ├── index.html     入口（按顺序加载 content/ 下 120+ 模块 → engine.js）
    ├── engine.js      规则引擎（回合循环 / 城市图 / 手机电脑 / 事件路由 / 死亡判定 / 渲染）
    ├── style.css      主题样式
    └── content/       ★模块化内容层（120+ 文件，可多 agent 并行填充）
        ├── core.js              核心层：属性体系 + 全局 helper + 基础数据
        ├── mvp-00-career.js     MVP 裁剪层：只保留「00 后职场」主线
        ├── city-map-career.js   成都城市俯瞰图（区域 / 设施 / 行动入口）
        ├── stage-manager.js     7 阶段主线脊柱
        ├── jobs.js / jobhunt-flow.js / companies.js   岗位库 / 求职面试流程
        ├── events-*.js          各类 VN 事件（职场链 / 通勤 / 家庭 / 恋爱 / 危机…）
        ├── destiny-*.js / main-arc-*.js / route-*.js   命运主线 / 出身主线 / 路线行动
        └── _assemble.js         装配层：把各模块组装成 window.CONTENT
```

## 给协作者 / AI 的话

- **加内容只改 `content/`**：各模块在全局作用域共享 `core.js` 的 helper，只需 `EVENTS.push(...)` 等注册即可，互不干扰、可并行填充。新增模块在 `_assemble.js` 之前插一行 `<script>`。
- **当前形态以代码为准**：`docs/GAME_DESIGN.md` 记录的是更早的跨世代设定，已被「成都 2026 · 00 后职场」收束，读时以 `mvp-00-career.js` + `stage-manager.js` + `city-map-career.js` 为准。
- 部署见 [`DEPLOY.md`](DEPLOY.md)；推到 GitHub `main` 即自动触发 Railway 重部署。
